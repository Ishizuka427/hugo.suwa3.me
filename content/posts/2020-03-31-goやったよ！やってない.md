brewでそれぞれインストール

```
$ brew install goenv
$ brew install direnv
```

goenvはバージョン管理なのでpyenvのようなもの  
direnvは環境変数の管理ツールで、移動がトリガーになるらしい。  
ローカル開発用ってかんじかな。

```
vi ~/.bash_profile
export PATH="$HOME/.goenv/bin:$PATH"
eval "$(goenv init -)"
```

バージョン指定してインストール

```
$ goenv install 1.11.1
$ goenv global 1.11.1
```

環境変数の編集

```
$ direnv edit .
direnv: $EDITOR not found.
```

ふむ。vimいれよう

```
$ export EDITOR=vim
$ direnv edit .
export GO111MODULE=on
export GOPATH=$(pwd)
export PATH=$PATH:$GOPATH/bin
```

```
$ go mod init .
```
